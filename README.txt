# Book Hike Pro — College Submission

**Type:** Frontend-only (HTML/CSS/JS) • No backend required  
**Features:**
- Unified Login/Sign Up (auth.html)
- Admin Panel: Add/Edit/Delete books, manage admins (localStorage)
- Product listing with search, category chips, sort
- Product details modal, Cart + Demo checkout
- Responsive design (mobile/tablet/desktop)
- Student details in footer (Bittu Barh, EV1, Roll 92400120400)

**Quick Start:** open `index.html` in your browser.

**Demo Admin:** `admin@bookhike.test` (any password) or create an account and add its email as admin from Admins section.
